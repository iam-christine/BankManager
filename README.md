## BankManager
Projet Java illustrant l’héritage avec différents types de comptes bancaires.

## Classes
- BankAccount (classe mère)
- CheckingAccount
- SavingsAccount
- COD

## Fonctionnalités
- Création de comptes
- Héritage des attributs account et balance
- Ajout d’attributs spécifiques

## Exécution
1. javac *.java
2. java Main
